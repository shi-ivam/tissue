##### To Run Dev Server :

1st Terminal

```
cd client && npm run build:watch && live-server  dist
```

2nd Terminal
```
cd server && nodemon
```
##### To Run Prod Server:

Run:
```
cd client && npm run build:prod && cd ../server && node index
```
